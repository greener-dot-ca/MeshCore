current page is dots at the bottom, current page larger dot
pages can be longer than the screen, visual indicators if there is more page above or below, scroll bar probably 
button 1 changes page forward, hold that button changes page backward
button 2 scrolls to next element, hold that button to action the element

elements can be selected and some acted upon: 
    toggle/on-off
    action
    next item in list of options

each element has a border when selected, some other form of visual indicator when not selected, actionable elements have an extra indicator (but clear that it is an element)
